# Prisoner Escape
My first own game designed :)


## Getting Started
### Running the project
Open the 'index.html' file in your browser.

## Controls:
* Left: A/left arrow key
* Right: D/right arrow key
* Jump: Up arrow key
* Switch grayscale: B 

## Built using
* JS
* WebGL
